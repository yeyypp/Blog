#流程控制语句
- go 只有一种循环结构， for 可以用for 代替while, 中间无需小括号
```
    for i := 0; i < 5; i++ {}
    for ; i > 5; i--{}
    代替 while
    for true {}
```
- if 无需小括号, 可以在条件表达式之前执行一个简单的语句，声明的变量作用域只在if中
```
    if x < 10 {}
    
    if i := Math.Pow(x, n); i < 10 {}
    
    func pow(a, b, c float64) bool {
    	if i := math.Pow(a, b); i < c {
    		return true
    	}
    	return false
    }
    
    func main() {
    	fmt.Println(
    		pow(3, 2, 10),
    		pow(3, 3, 20),
    	)
    }
```
