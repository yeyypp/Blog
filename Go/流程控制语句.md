# 流程控制语句

### switch

执行顺序是从上到下依次执行，直到匹配成功，停止。
```
package main

import "fmt"

func main() {
	switch "a" {
	case "b":
		fmt.Println("b")
	case "a":
		fmt.Println("a")
	default:
		fmt.Println("default")
	}
}
```

没有条件的switch 如if true
```
package main

import "fmt"

func max(a, b int) int {
	switch {
	case a < b:
		return b
	case a > b:
		return a
	default:
		return -1
	}
}

func main() {
	fmt.Println(max(1, 2))
}
```
