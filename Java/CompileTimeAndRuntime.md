# Compile time vs Runtime

- Compile time

    It is the time that the source code convert to executable file
    
    Input is the source code  
    Output is an executable file
    
    compile time error:
    
    - Syntax errors
    - Typechecking errors
    - Compiler crashes
    
- Runtime
    
    It is the time that the system running the executable file.
    
    runtime exceptions:
    
    - Division by zero
    - File not found
    - null pointer
